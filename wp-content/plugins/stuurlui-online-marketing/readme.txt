Custom settings for customers of Stuurlui. Allows for easy manageability of Wordpress sites built by Stuurlui Online Marketing. WARNING: will disable auto updates


== Description ==

This plugin adds and changes a number of Wordpress settings.

WARNING: this plugin disables auto-updates by default!

It allows for the following functionalities:

* Changes login screen
* Checks if important plugins are installed
* Allows adding of fontawesome


Wishlist:

* Admin pages with tabs
* Nicer maintenance message (http://wordpress.stackexchange.com/questions/75806/custom-maintenance-page)


== Installation ==

Install plugin via the repository or manually.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/stuurlui-online-marketing` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Stuurlui screen to configure the plugin


== Frequently Asked Questions ==

= What should I do with this plugin? =
Nothing. It is for Stuurlui employees only! You can use it without issues, but please be advised: This plugin will disable auto-updates of Wordpress, themes and plugins by default.

= I have a question, who do I contact? =
Send us an email at ahoy@stuurlui.nl with your question(s).
